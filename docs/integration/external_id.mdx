# External id
To retrieve the information you need to send an external identifier for the donation transaction.  

More than one donation per external identifier is not allowed scoped by integration.

## How to send external identifier
To send the identifier just put it as a parameter in the donation link.

ex: https://dapp.ribon.io/asdasdasd
